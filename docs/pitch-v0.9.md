# AXIOM v0.9 Pitch

## One-liner

AXIOM is a trust kernel for AI systems.

## Short pitch

Models generate. Agents act. Memory stores. AXIOM judges.

## Problem

AI systems can generate, retrieve, and act, but they often cannot explain whether a claim is actually safe to believe.

## Solution

AXIOM verifies claims against a local graph, provenance, semantic risk signals, contradiction rules, and reasoning traces.

## Moat

- Local-first graph
- Causal reasoning
- Trust Kernel + ATP
- Semantic Trust Gate
- Claim decomposition
- Reasoning trace
- Deterministic tests

## Positioning

- Not a model
- Not a thin wrapper
- Not a generic memory store
- A deterministic trust and reasoning layer that sits around models, memory, tools, and agents
