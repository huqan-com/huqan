# GTM Static Demo Landing

## Deployment options

- Vercel
- GitHub Pages
- Cloudflare Pages

## Recommended first deploy

Use a Vercel preview deploy first. It is the fastest way to validate the static landing and share a reviewable URL.

## Recommended stable domain

Later, move the public demo to `demo.huqan.com`.

## Demo scope

This landing is static and backend-free. It presents the HUQAN / AXIOM narrative, the Trust Receipt concept, and a small frontend-only simulation.

## Notes

- no live engine runs in the browser
- no API keys
- no telemetry
- no analytics unless explicitly approved
- no change to trust kernel, verifier, or MemoryStore
