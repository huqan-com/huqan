Deterministic verification

AXIOM evaluates claims with stable, testable rules. The landing page points to the same docs the repo uses to keep the kernel bounded and reviewable.

Canonical identity, not truth inflation

Entity resolution expands lookup keys only when there is a valid domain or canonical evidence. It does not manufacture verdicts or rewrite the original literal.

Write-side anchoring

When metadata is present, the graph persists the resolved entity anchor through JSON and SQLite roundtrips without changing the underlying graph contract.
