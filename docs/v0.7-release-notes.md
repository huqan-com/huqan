# AXIOM v0.7.0 Release Notes

Branch: `main`

This release notes file covers the final promotion from `v0.7.0-rc.1` to `v0.7.0`.

## Scope

AXIOM v0.7 is the causal reasoning line. It focuses on deterministic symbolic causal reasoning instead of probabilistic prediction or UI expansion.

Included:

- causal relations: `CAUSES`, `PREVENTS`, `ENABLES`, `DEPENDS_ON`, `LEADS_TO`
- deterministic causal traversal
- what-if simulator
- causal finalizer summary output
- deterministic demo scenario for `autoLearn default true`

## Smoke Checklist

Run the following before publishing or verifying the final `v0.7.0` tag:

```bash
npm test
node demo-causal-autolearn.js
node --test graph.causal.test.js causalSimulator.test.js finalizer.causal.test.js demo-causal-autolearn.test.js
```

## Known Limitations

- Not a full world model
- No probabilistic prediction layer
- No dedicated causal UI yet
- No enterprise governance layer
- No multi-user permissions or audit framework
- Causal relations are structured inputs, not autonomous discovery

## Release Notes

- `v0.7.0` is the final release for the causal reasoning line.
- `v0.7.0-rc.1` remains the release candidate marker for the branch history.
- The causal line has been promoted to `main`.
- Existing `v0.6.0` release artifacts remain available in release history.
